CSCI 4208 HW4

For this assignment, I created a safari sightseeing game where the user can click a button to view pictures and information about different animals.

In order for the animal generation to occur I used the zoo animals api that can be found at https://zoo-animal-api.herokuapp.com

This assignment was used to demonstrate the usage of a Client REST API.
